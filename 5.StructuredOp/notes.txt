1) To get structured data from models that support it, simply use with_structured_output.
2) We can define the structure using the following methods:
-> Typed Dictionary => only for representational purpose, not for data validation
-> Pydantic => provides data validation
-> JSON Schema

3) with_structured_output can take an optional argument called method. Possible values for method can be function_calling, json_mode, json_schema JSON mode. Function calling is generally most reliable for production.
